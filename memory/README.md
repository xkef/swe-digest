# Memory

Public memory for recurring digest context.

Use this directory for compact facts that help future daily runs interpret new stories. Keep it public-safe. Do not store private employer details, private plans, account data, secrets, or personal contact data.

Files:

- `profile.md`: public-safe preference profile.
- `followups.md`: open story threads that need later checks.
- `entities.md`: recurring projects, companies, people, and standards.
- `source-reliability.md`: notes on source quality.
