SWE Digest is a daily public record of software engineering signal.

It tracks platform changes, security issues, outages, AI releases, developer tools, language releases, technical blog posts, company events, Hacker News discussion, Reddit hype, and follow-up threads that need memory across days.

## Reading model

- `Digests`: dated daily summaries.
- `Sources`: collection map and source priorities.
- `Search`: full-site search across digest text.
- `Feed`: Atom feed for daily reading.

## Editorial model

Each story links to primary sources first. Discussion links are labeled as discussion. Rumors stay marked as rumors. Developing incidents stay marked as developing. Memory files track recurring topics and follow-ups without turning the digest into a link dump.
