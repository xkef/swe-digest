## Source priority

1. Primary source: official release note, incident report, advisory, filing, repository release, maintainer post.
2. Technical context: engineering blog, postmortem, benchmark with reproducible detail, conference talk, maintainer interview.
3. Discussion: Hacker News, Reddit, Lobsters, YouTube commentary, podcasts.
4. Market reporting: Reuters, Bloomberg, TechCrunch, CNBC, Financial Times, Wall Street Journal, official newsroom.

## Daily source classes

- Hacker News front page, Ask HN, Show HN, and targeted Algolia searches.
- Reddit hot and top daily posts for programming, AI, security, infrastructure, and tool subreddits.
- AI vendor blogs, docs, changelogs, status pages, and model cards.
- Security advisories from CISA, NVD, OSV, GitHub Security Advisories, vendors, and maintainers.
- Status pages for cloud providers, developer platforms, package registries, identity providers, payments, and observability vendors.
- GitHub releases and changelogs for watched repositories.
- Engineering blogs with implementation detail or incident analysis.
- Company news for acquisitions, IPOs, filings, governance, and licensing changes.
- YouTube RSS feeds when videos add context to written sources.
